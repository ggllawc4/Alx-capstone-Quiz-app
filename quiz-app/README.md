# Quiz App

A simple React-based quiz app that fetches trivia questions from the Open Trivia Database API.

## Features
- Multiple-choice questions
- Dynamic scoring
- Responsive UI built with Tailwind CSS

## Tech Stack
- React
- Tailwind CSS
- Open Trivia Database API
